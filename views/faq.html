FAQ Documentation

What is Aurelius?

Aurelius is an API version of Marcus Aurelius' Meditations.

Why did you build this?

Aurelius originated as a project for Appboy Hackday 2017. It was inspired by an observation from a friend that Meditations is best read in small chunks.

Which translation of Meditations does this use?

Aurelius uses a modified version of George Long's 1862 translation, which is in the public domain. Archaisms in that translation have been replaced with more modern English words. For example, "thou" has been replaced by "you", "livest" has been changed to "live, and etc.
